# BiliRanking
哔哩哔哩数据分析整理，月刊鬼畜榜数据及图片专用生成工具

# 亮点
* 二维码登录获取Cookie  
* 通过API获取并算分后的数据可保存为.sdyg（圣地亚哥数据库）格式
* 一键批量下载封面、视频
* 更多……
